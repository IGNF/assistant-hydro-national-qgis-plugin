# Assistant hydro national

Ce plugin est une aide à la modification sémantique des tronçons hydrographique

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)

## Fonctionnalités

French : 
* Cette interface permet de modifier certains champs des tronçons hydrographiques.


English :
* This interface allows users to modify certain attributes of hydrographic segments.
